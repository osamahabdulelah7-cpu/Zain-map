# Zain Mapper Pro

محرك احترافي لتحليل ملفات WiGLE وتحديد المواقع الحقيقية لراوترات Wi-Fi، مع التركيز على شبكات Zain.

## المميزات (قيد التطوير)

- قراءة ملفات CSV و CSV.GZ الخاصة بـ WiGLE
- استخراج شبكات Zain فقط
- دمج شبكات 2.4GHz و 5GHz
- إزالة التكرارات
- تحديد الشركة المصنعة من عنوان MAC (IEEE OUI)
- تحليل فترة الوقوف لتحديد موقع الراوتر
- إزالة القراءات الشاذة
- إنشاء ملفات KML لـ Google Earth
- إنشاء خرائط HTML
- تصدير Excel وPDF
- مقارنة نتائج عدة جولات مسح
